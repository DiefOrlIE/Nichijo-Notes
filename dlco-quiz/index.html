第0章 导论~2分

0.1填空
计算系统软件层次自顶向下：Application → ______ → Assembler → ______（接口层）。
参考答案
编译器/操作系统(Compiler / OS)；ISA（指令集体系结构）。 ISA 是软硬件分界面：往上是软件，往下是微体系结构→数字设计→电路→晶体管。
0.2解释
ISA 与微体系结构（Microarchitecture）有什么区别？
参考答案
ISA 是规范/契约（指令集、寄存器组织、寻址方式、数据类型等程序员可见的接口）；微架构是 ISA 的具体硬件实现。同一 ISA 可有多种微架构（如不同性能的 x86 处理器）。 易错：把两者混为一谈。ISA 不变，微架构可演进。

第1章 二进制编码~5分

1.1填空
冯·诺依曼结构五大部件：运算器、控制器、______、______、______。其核心思想是______。
参考答案
存储器、输入设备、输出设备；存储程序（指令与数据均以二进制存于同一存储器，不加区分，自动逐条执行）。
1.2填空
写出 PC、IR、MAR、MDR 各自存放什么。
参考答案
PC=下一条指令地址；IR=当前执行的指令；MAR=访存地址；MDR=与存储器交换的数据。
1.3填空
n 位定点整数补码定义 [X]补 = ______（mod 2ⁿ），可表示范围为 ______。求负数补码的规则是 ______。
参考答案
2ⁿ + X；范围 [−2ⁿ⁻¹, 2ⁿ⁻¹−1]；各位取反、末位加 1。 补码 0 唯一；模是 2ⁿ 不是 2ⁿ−1；真值公式 X = −aₙ₋₁·2ⁿ⁻¹ + Σ aᵢ·2ⁱ。
1.4解释
为什么 IEEE 754 浮点数的尾数用原码、阶码用移码？
参考答案
尾数原码：符号与数值分离，便于乘除（符号异或、数值无符号运算）。阶码移码：移码 = 真值 + 偏置，使阶码可直接按无符号整数比较大小，简化对阶与排序。
1.5填空
IEEE 754 单精度：S(1) + 阶码 E(8, 偏置 ______) + 尾数 M(23)。规格化数真值 = ______，此时 E 的取值范围是 ______。阶码全 0、尾数非 0 表示 ______。
参考答案
偏置 127；真值 (−1)ˢ × 1.M × 2^(E−127)；E ∈ [1, 254]；非规格化数（隐含位为 0，指数固定 −126）。
1.6仿真题
小端存储，地址 0x100–0x103 依次存 63H、73H、61H、69H。（已知 'a'=61H）
(1) 作为无符号整数，右移 24 位后的十进制值 = ______；(2) 作为字符串 = ______。
参考答案
小端：低地址存低字节 → 整数 = 0x69617363。 (1) 右移 24 位取最高字节 = 0x69 = 105； (2) 字符串按地址从低到高逐字节：63='c'、73='s'、61='a'、69='i' → "csai"。 字符串顺序按地址递增；整数拼接时高地址是高位字节——别搞反。
1.7解释
大端与小端的区别？C 语言中 -1 < 0U 的结果是什么，为什么？
参考答案
大端=最高有效字节存低地址；小端=最低有效字节存低地址。 -1 < 0U 为 假：有符号与无符号混合运算时，有符号数被转成无符号数，−1 变成 0xFFFFFFFF（很大），故不小于 0U。

第2章 数字逻辑基础~7分

2.1填空
德·摩根定理：¬(X·Y) = ______，¬(X+Y) = ______。反演定律求补函数的规则中，对「非单个变量上的整体取反号」应如何处理？
参考答案
¬X + ¬Y；¬X · ¬Y；整体取反号 保留不变（不可拆成单变量反）。
2.2填空
分配律有两条：X·(Y+Z) = ______；X + Y·Z = ______（与算术不同）。一致律（冗余项定理）：XY + ¬XZ + YZ = ______。
参考答案
XY + XZ；(X+Y)(X+Z)；XY + ¬XZ（YZ 是冗余项，消去可避免冒险）。
2.3仿真题
F(A,B,C,D) = Σm(1,3,5,7,9,11,13,15)。(1) 用最大项表示 F = ΠM(______)；(2) 序号最大的最大项 M₁₄ 的逻辑表达式 = ______。
参考答案
(1) 最小项与最大项编号互补 → ΠM(0,2,4,6,8,10,12,14)； (2) M₁₄：14=1110，最大项是和项、变量为 1 取反变量、为 0 取原变量 → ¬A+¬B+¬C+D。 本题 F 其实就是「D=1」（所有奇数项）。
2.4填空
直流噪声容限：高态 NM_H = ______，低态 NM_L = ______。CMOS 动态功耗 P_D = ______。
参考答案
NM_H = V_OHmin − V_IHmin；NM_L = V_ILmax − V_OLmax；P_D = (C_L + C_PD)·V_CC²·f。 易错：NM_H/NM_L 的减数被减数顺序常记反。
2.5解释
CMOS 与非门和或非门的上拉（PMOS）/下拉（NMOS）网络结构分别是怎样的？
参考答案
与非门：PMOS 并联（上拉）+ NMOS 串联（下拉）。或非门：PMOS 串联 + NMOS 并联。 规律：与→下拉串联，或→下拉并联；上拉与下拉对偶。严禁 VDD 到地出现低阻通路。
2.6填空
卡诺图化简：相邻 2ⁱ 个「1 格」合并消去 ______ 个变量。质蕴涵项、实质蕴涵项、最小覆盖的关系是 ______。
参考答案
消去 i 个变量；最小覆盖 = 所有实质蕴涵项（含独有最小项的质蕴涵项）+ 最少数量的其他质蕴涵项以覆盖全部最小项。
2.7解释
如何用卡诺图判断电路是否存在静态冒险？如何消除？（去年简答原题）
参考答案
判断：卡诺图上两个为 1 的质蕴涵圈「相切」（相邻但未被同一个圈覆盖），当输入在这两圈间跳变、仅一个变量变化时，因路径延迟差可能产生毛刺 → 存在静态-1 冒险。 消除：在与或式中加一个冗余的一致项（把相切的两圈搭起来的圈），覆盖跳变边界即可。

第3章 组合逻辑电路~5分

3.1填空
组合逻辑电路必须同时满足的三大构成规则：______、______、______。
参考答案
①每个组件本身是组合电路；②任两组件输出端不能直接互连；③无反馈（输出不能直接或间接接回输入）。
3.2填空
全加器：S = ______，Cout = ______。n 位行波进位加法器的缺点是 ______。
参考答案
S = A ⊕ B ⊕ Cin；Cout = AB + ACin + BCin；缺点：高位需等低位进位逐级传递，速度慢，延迟随位数线性增长。
3.3解释
74X138（3-8 译码器）的使能逻辑是怎样的？输出是高有效还是低有效？
参考答案
三个使能端 G1（高有效）、G2A_L、G2B_L（低有效），内部使能 EN = G1·¬G2A_L·¬G2B_L，三者同时满足才工作；输出低有效（Yi_L = ¬(EN·Mi)）。
3.4解释
用降维法、以 B、C 为选择端的 4 选 1 MUX 实现 F(A,B,C)=Σm(1,3,5,6)，写出四个数据输入 D00、D01、D10、D11。
参考答案
按 BC 分组看 A：
D00(BC=00, m0/m4)=0；D01(BC=01, m1/m5)=1；D10(BC=10, m2/m6)=A（仅 m6=ABC̄有效）；D11(BC=11, m3/m7)=¬A（仅 m3 有效）。
3.5填空
组合电路时序：传输延迟 Tpd 指 ______；最小延迟 Tcd 指 ______。三态门 EN=0 时输出为 ______。
参考答案
Tpd=输入变化到所有输出稳定的最长时间（沿关键路径）；Tcd=输入变化到任一输出首次变化的最短时间；EN=0 输出为高阻态 Hi-Z。

第4章 时序逻辑电路~22分 · 重点

4.1解释
锁存器和触发器的异同？（去年简答原题）
参考答案
同：都是双稳态、存 1 位、靠反馈环路记忆。异：锁存器电平触发（有效电平期间透明，输入直通输出）；触发器边沿触发（仅时钟沿采样），抗干扰强、便于同步设计。
4.2填空
Mealy 型输出 = ______；Moore 型输出 = ______。状态图上区别：Moore 输出标在 ______，Mealy 标在 ______。
参考答案
G(现态, 输入)；G(现态)；Moore 标在状态圈里（如 C/10），Mealy 标在转移箭头上。
4.3填空
写出特征方程：D 触发器 Q*=______；JK 触发器 Q*=______；T 触发器 Q*=______。SR 锁存器的约束条件是 ______。
参考答案
Q*=D；Q*=J·Q̄ + K̄·Q；Q*=T⊕Q；约束 S·R≠1（S、R 不能同时为 1）。
4.4填空
同步时序电路两条时序约束：最小时钟周期 t_clk > ______；最小保持 t_hold < ______。建立时间容限 = ______（需 >0）。（去年填空：时钟周期 > 建立时间 + ___ + ___ 的和）
参考答案
t_clk > t_ffpd(max) + t_comb(max) + t_setup；t_hold < t_ffpd(min) + t_comb(min)；建立容限 = t_clk − t_ffpd(max) − t_comb(max) − t_setup。 去年那两空 = Clk-to-Q 延迟(t_ffpd) 和 组合逻辑延迟(t_comb)。max 用于建立约束、min 用于保持约束，别混。
4.5填空
状态图的「完备性」指 ______；「互斥性」指 ______。状态编码相邻三准则中的第一条是 ______。
参考答案
完备性：从每个状态出发的所有转移条件逻辑或 = 1（任何输入都有去向）；互斥性：各转移条件两两逻辑与 = 0（不会同时满足两条）；第一准则：次态相同的现态，编码尽量相邻。
4.6解释
什么是未用状态的「自启动」？设计 FSM 时为什么要做未用状态分析？
参考答案
编码空间大于实际状态数时会有未用状态。把未用状态当现态代入激励方程算其次态，若能在有限个时钟内回到正常工作循环且无错误输出，则电路能自启动；否则可能「挂起/死锁」。因此必须检查，必要时用预置或无关项修正。
4.7仿真题
给定一个状态转移图（如 5 状态、每状态 2 位输出的 Moore 机），写出同步时序电路的设计步骤（五步），并说明第(2)问「时序分析」和第(3)问「未用状态分析」各怎么做。（去年大题原型）
参考答案
设计五步：①状态编码（5 态需 3 位）→ ②选触发器（一般 D，激励=次态）→ ③列状态转移真值表 → ④卡诺图求各 Dᵢ 激励方程与输出方程 → ⑤画电路图。 (2) 时序分析（反推）：写驱动方程→代特征方程得次态方程→列状态表/画状态图，验证与题目一致。 (3) 未用状态分析：把未用编码（如 101/110/111）代入方程求次态，确认能否回到有效状态环（自启动）。
4.8解释
寄存器堆（Register File）的读操作和写操作在时序上有何本质区别？
参考答案
读是组合逻辑：给地址经 MUX 直接输出，不需时钟。写是时序逻辑：需时钟沿 + 写使能 WE，地址/数据/WE 须满足建立时间，在时钟沿写入。
4.9解释
异步行波加法计数器与减法计数器，在触发器的时钟连接上有什么区别？
参考答案
以下降沿触发的翻转触发器为例：加法用前级 Q 驱动后级时钟（Q 由 1→0 时进位）；减法用前级 Q̄ 驱动后级时钟。把 Q 与 Q̄ 对调即在加/减之间切换。

第6章 运算方法和运算部件~17分 · 重点

6.1填空
补码减法 [A−B]补 = ______。带符号溢出 OF = ______（用进位表示）；无符号溢出标志 CF = ______；符号标志 SF = ______。
参考答案
[A]补 + [−B]补（[−B]补 = ¬[B]补 + 1）；OF = Cₙ ⊕ Cₙ₋₁（最高位与次高位进位不同）；CF = Cout ⊕ Cin（无符号进位/借位）；SF = 结果最高位 Fₙ₋₁。
6.2填空
用减法做大小比较：无符号数 A>B 的条件是 ______；带符号数 A>B 的条件是 ______。
参考答案
无符号：CF=0 且 ZF=0；带符号：SF=OF 且 ZF=0（即 A<B 时 SF≠OF）。
6.3填空
先行进位加法器（CLA）：进位生成 Gᵢ = ______，进位传递 Pᵢ = ______，进位递推 Cᵢ = ______。它相比行波进位的优势是 ______。
参考答案
Gᵢ = AᵢBᵢ；Pᵢ = Aᵢ+Bᵢ（或 Aᵢ⊕Bᵢ）；Cᵢ = Gᵢ + Pᵢ·Cᵢ₋₁；优势：并行算出各位进位，消除逐级等待，延迟与位数基本无关。
6.4仿真题
推导补码两位乘法（Booth 两位乘法）的规则：根据乘数相邻三位 yᵢ₊₁ yᵢ yᵢ₋₁，部分积该加什么、移几位、循环几次？列出编码表。（去年大题原型）
参考答案
看 yᵢ₊₁yᵢyᵢ₋₁ 决定加 {0, ±X, ±2X}，再算术右移 2 位，循环 n/2 次：
000/111→+0；001/010→+X；011→+2X；100→−2X；101/110→−X。 原理：Booth 一位乘 Pᵢ₊₁=[Pᵢ+(yᵢ₋₁−yᵢ)X]·2⁻¹，合并相邻两步即得。「+3X 麻烦」改成本次 −X、下次 +X（下次已右移 2 位等价 +4X）。硬件仍只用加法器+移位器。
6.5解释
原码两位乘法为什么需要采用 3 位符号位（模 8 补码）？
参考答案
两位乘每次部分积右移 2 位，且会出现 −X（处理 +3X=−X+4X）操作；2 位符号位不足以容纳右移与负部分积可能的跨位，需 3 位符号位防止部分积移位时符号/数值错位。
6.6填空
补码乘法的溢出判断：乘积高 ______ 位为全 0 或全 1 时不溢出。无符号乘法与补码乘法的「低 n 位结果」关系是 ______。
参考答案
高 n+1 位全 0 或全 1（等价于高 n 位每位都等于低 n 位的符号位）；两者低 n 位结果完全相同。
6.7解释
不恢复余数法（加减交替法）做除法的上商规则是什么？（区分原码除法和补码除法）
参考答案
原码除（看余数正负）：R≥0 商 1、下步 2R−Y；R<0 商 0、下步 2R+Y；末余数为负要加回除数修正。 补码除（看余数 R 与除数 Y 符号异同）：同号商 1、下步 2R−Y；异号商 0、下步 2R+Y；末位恒置 1；X、Y 异号时商末位再加 1 修正。
6.8填空
移码加减运算：移码的和/差等于「和/差的 ______」，运算后需将结果的 ______ 取反还原。负数除以 2ᵏ 为实现向零舍入，应先做 ______ 再右移。
参考答案
等于「和/差的补码」；运算后将结果符号位（最高位）取反；负数除 2ᵏ 先做 (x + 2ᵏ − 1) 纠偏再算术右移。 易错：负数 x>>1 ≠ x/2，移位向负无穷舍入、除法向零舍入。

第7章 指令系统~27分 · 最大头

7.1填空
写出有效地址 EA：直接 ______、间接 ______、寄存器间接 ______、相对 ______、基址 ______、变址 ______。寄存器间接寻址中，操作数寄存器里存放的是 ______。
参考答案
直接 EA=A；间接 EA=(A)；寄存器间接 EA=(R)；相对 EA=A+(PC)；基址 EA=A+(B)；变址 EA=A+(I)；寄存器间接里存的是操作数的地址。
7.2仿真题
扩展操作码：指令字长 16 位、地址码 6 位。已有二地址指令 15 条、一地址指令 34 条，问零地址指令最多有多少条？写出算式。
参考答案
二地址用 4 位操作码（剩 2×6=12 位地址），可用 2⁴=16，用掉 15，剩 1 → 扩展给一地址（再用 6 位）：1×2⁶=64，用掉 34，剩 30 → 扩展给零地址（再用 6 位）： ((2⁴−15)·2⁶−34)·2⁶ = 30 × 64 = 1920 条。 核心：逐级把没用完的短码空间×下一字段的编码数，扣掉已占用的。
7.3填空
RISC-V 32 位六种指令格式：R、I、S、B、U、J。其中立即数 12 位的是 ______，放高 20 位的是 ______，用于分支的是 ______，用于 jal 的是 ______。
参考答案
12 位立即数：I 型、S 型（B 型为 13 位含末 0）；高 20 位：U 型；分支：B 型；jal：J 型（21 位含末 0）。
7.4仿真题
RISC-V 综合题：(1) 给定一个 opcode，R 型指令理论上最多多少条？I 型呢？(2) I 型立即数下标是 imm[11:0]，U 型是 imm[31:12]，分析两者设计思路与扩展方式的不同。
参考答案
(1) R 型靠 funct3(3 位)+funct7(7 位)区分 → 2³×2⁷ = 2¹⁰ = 1024 条；I 型只靠 funct3 → 2³ = 8 条。 (2) I 型：12 位小立即数放低位，运算时符号扩展到 32 位（配合 addi/load 的近距偏移）；U 型：20 位放高位 [31:12]，低 12 位补 0（=左移 12 位），不符号扩展（lui 专门拼高 20 位）。二者 lui+addi 合用才能凑任意 32 位常数。
7.5填空
B 型、J 型立即数的共同特点是最低位恒为 ______，使用时要先 ______ 再 ______ 得到字节偏移。jal 的跳转范围约 ______，分支约 ______。
参考答案
最低位恒为 0（半字/字对齐）；先符号扩展再左移 1 位；jal 约 ±1 MiB（21 位有符号），分支约 ±4 KiB（13 位）。
7.6解释
RV32I 能不能只用一条指令把任意 32 位操作数写入寄存器？为什么？（去年综合题第 5 问）
参考答案
不能（部分同意该说法）：12 位以内立即数可用一条 addi rd, x0, imm；但 32 位指令字装不下完整 32 位立即数（还要留 opcode、rd），任意 32 位常数必须 lui+addi 两条（高 20 位 + 低 12 位）。
7.7解释
用 LUI + ADDI 加载一个低 12 位最高位为 1 的 32 位常数时，为什么要用「hi+1」修正？举例。
参考答案
ADDI 的 12 位立即数符号扩展，若低 12 位最高位为 1 则被当成负数，会使高 20 位少 1。故先把高位装 hi+1，再用 ADDI 加一个负的 12 位立即数抵消。 例：装 0x1FFF → LUI x5,2; ADDI x5,x5,-1。
7.8填空
CISC vs RISC：访存方式上 RISC 是 ______ 型（只有 ______ 指令访存）；控制方式 RISC 多用 ______；指令周期 RISC 多为 ______。
参考答案
Load/Store 型（只有 Load/Store 访存，运算指令只碰寄存器）；控制用组合逻辑/硬布线；多为单周期（便于流水线）。
7.9解释
简述一条指令的执行过程；异常和中断如何处理？（去年简答原题）
参考答案
执行过程：取指（IR←M[PC]，PC←PC+4）→ 译码 → 取操作数 → 执行(ALU) → 写回 → 循环。 异常/中断处理：①关中断；②保护断点和程序状态（EPC←断点，Cause←原因编码）；③转入处理程序；处理完恢复现场、返回。异常来自 CPU 内部（溢出/非法指令），中断来自外部（I/O）。
7.10解释
RISC-V 调用约定里「调用者保存（caller-saved）」和「被调用者保存（callee-saved）」分别指哪些寄存器，含义是什么？
参考答案
caller-saved（临时 t0–t6、参数 a0–a7）：被调函数可随意改，调用方若需保留要自己先存。callee-saved（s0–s11）：被调函数若要用必须先保存、返回前恢复，保证对调用方不变。

第8章 中央处理器~18分 · 重点

8.1填空
CPU = ______ + ______。数据通路里「操作元件」是 ______ 逻辑（举例 ______），「状态元件」是 ______ 逻辑（举例 ______）。
参考答案
数据通路 + 控制器；操作元件=组合逻辑（ALU、MUX、加法器、译码器）；状态元件=时序逻辑（PC、寄存器堆、存储器）。
8.2填空
CPU 性能核心公式：CPU 时间 = ______ × ______ × ______。CPI = ______。影响时钟频率的两个因素是 ______。
参考答案
CPU 时间 = 指令条数 × CPI × 时钟周期；CPI = 总时钟周期数 / 指令条数（综合 CPI=Σ CPIᵢ·Fᵢ）；时钟频率受微体系结构和工艺影响。
8.3仿真题
指令 jal rd, imm20，功能 PC←PC+SEXT(imm[20:1]≪1); R[rd]←PC+4。结合 J 型数据通路：(1) 描述执行过程；(2) 写出关键控制信号取值；(3) 转移目标地址范围多大？（去年大题原型）
参考答案
(1) 取指→译码识别 J 型→扩展器对 imm 符号扩展并左移 1 位；一路加法器算 PC+offset 经下地址逻辑送 PC，另一路算 PC+4 经 MUX 写回 rd。 (2) RegWr=1、Jump=1、Branch=0、MemWr=0、MemtoReg→选 PC+4、ExtOp=immJ、ALU 做加；ALUASrc 选 PC、ALUBSrc 选常数 4。 (3) 偏移 21 位有符号、末位 0 → 范围 PC−2²⁰ ~ PC+(2²⁰−2) 字节（约 ±1 MiB）。
8.4填空
五种立即数扩展：immI = ______（位数与扩展方式）；immU = ______；immB 的最低位强制为 ______。逻辑指令 ori 的立即数是 ______ 扩展。
参考答案
immI：12 位符号扩展到 32 位；immU：高 20 位放 [31:12]、低 12 位补 0（非符号扩展）；immB 最低位强制为 0；ori 立即数也是符号扩展（易错点，不是零扩展）。
8.5解释
单周期 CPU 的时钟周期为什么由 lw 指令决定？它的关键路径包含哪些环节？
参考答案
单周期所有指令共用一个周期，须容纳最慢指令。lw 经过的环节最多：PC → 指令存储器 → 读寄存器 → ALU 算地址 → 数据存储器读 → 写回寄存器(setup)（再加时钟偏移）。所以周期由 lw 关键路径定，简单指令被迫等长周期 → 性能浪费。
8.6填空
取指与 PC 更新的逻辑顺序必须是 ______。单周期下地址逻辑：Jump=1 选 ______；Branch=1 且 Zero=1 选 ______；否则选 ______。
参考答案
必须先取指（Instr←M[PC]）再 PC←PC+4，顺序不可颠倒；Jump 选跳转目标，Branch&Zero 选分支目标，否则选 PC+4。
8.7解释
流水线三大冒险分别是什么？各自的典型解决方法？
参考答案
结构冒险：部件争用 → 指令/数据 Cache 分离、固定流水段使用。数据冒险：ALU 相关用转发(Forwarding)；Load-use 需阻塞 1 周期+转发。控制冒险：转移目标未定 → 清空错误指令、分支预测（静态/动态）、延迟分支。
8.8解释
异常和中断的区别？异常处理中 EPC 保存的「断点」有哪两种情况？
参考答案
异常来自 CPU 内部（溢出、非法指令、缺页），中断来自外部（I/O 请求）。EPC：故障类异常存 PC−4（出错指令本身，便于重执行），自陷/中断存 PC（下一条）。Cause 寄存器记录原因。
8.9填空
多周期处理器 CPI = ______。微程序控制器中，水平型微指令的特点是 ______，垂直型是 ______。
参考答案
CPI = Σ(指令占比ᵢ × 指令周期数ᵢ)（按频率加权）；水平型并行度高、微程序短、字长长；垂直型编码效率高、易编制、速度慢。
8.10解释
为什么说流水线不能缩短「单条指令」的执行时间？它提升性能的本质是什么？
参考答案
单条指令仍要走完所有流水段（甚至因段对齐略变长）。流水线靠多条指令重叠执行提高吞吐率：理想下每个周期完成一条，加速比≈流水级数（受冒险和流水寄存器开销限制）。瓶颈在最慢的那一段。
