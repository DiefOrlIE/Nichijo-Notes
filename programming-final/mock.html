Q1简答基础

解释型语言与编译型语言的执行过程有何不同？比较两者的运行效率差异。

参考答案
编译型（C/C++）：运行前由编译器把整份源码一次性翻译成机器码、链接成可执行文件，运行时脱离编译器直接执行。
解释型（Python）：由解释器在运行时逐行翻译并执行，每次运行都需解释器在场。
效率：编译型把翻译成本前置，运行时是纯机器码，更快；解释型边翻译边执行有额外开销，较慢，但即改即运行、便于调试、跨平台（源码+解释器即可）。
Q2简答基础

解释字符串、列表、元组的含义，并指出其中哪些是可变对象。

参考答案
字符串 str：不可变的字符序列；列表 list：可变的有序序列，能增删改；元组 tuple：不可变的有序序列。
三者中只有 list 可变，str、tuple 不可变。意义：不可变对象可做字典键/集合元素（可哈希），对它"修改"会新建对象（id 变）；可变对象原地修改（id 不变）。
Q3简答中等

分别解释纯函数和高阶函数的含义，并判断一个函数能否同时是纯函数和高阶函数。

参考答案
纯函数：返回值只依赖输入参数、且无副作用（不改变外部状态），相同输入永远同样输出。
高阶函数：以函数为参数，或返回函数。
能同时是：两者约束的维度不同——"纯"管有无副作用，"高阶"管是否传入/返回函数，互相正交。例 compose(f,g)=lambda x:f(g(x))：接收并返回函数（高阶），又不碰外部状态（纯）。
Q4简答中等

哪些行为会产生函数副作用？函数副作用会带来哪些影响？

参考答案
产生副作用的行为：修改全局变量（或用 global/nonlocal 改外层绑定）、修改传入的可变对象（list/dict/对象属性）、I/O（print、读写文件、读键盘、网络）。
影响：函数不再自包含，行为依赖/改变隐藏状态，导致难以推理、难测试、难并发，且 bug 难复现（同样输入结果可能不同）。工程上应把副作用收敛到少数边界。
Q5简答基础

分别解释抽象和封装的含义，并说明两者的异同。

参考答案
抽象：从使用者视角，只关心对象"能做什么"、忽略"怎么实现"（关注接口/行为）。
封装：从实现者视角，把数据与实现细节隐藏，只暴露受控接口，防止外部乱改内部状态（关注隐藏/保护）。
同：都为控制复杂度，都区分对外接口与内部细节。异：抽象决定"该露什么"，封装负责"把不该露的藏好"。
Q6简答中等

说明 PCA 的主要思想及实现步骤。

参考答案
思想：找一个线性变换把 D 维数据投影到 d 维（d<D），使投影后方差最大（沿信息最丰富的方向建坐标系，尽量保留信息），等价于重构误差最小。
步骤：① 中心化（各特征减均值）；②（尺度差异大时）z-score 规范化；③ 算协方差矩阵 C=(1/m)XᵀX；④ 对 C 特征值分解；⑤ 按特征值从大到小取前 d 个特征向量组成 W；⑥ 投影 Z=X·W。
要点：必须中心化；特征值=该方向方差，越大越重要；方差贡献率 λ_i/Σλ 用于选 d。
Q7简答基础

NumPy 中视图（view）与副本（copy）有何区别？哪些索引返回视图、哪些返回副本？

参考答案
视图与原数组共享底层数据，改视图会改原数组；副本是独立数据，互不影响。
基本切片（a[1:3]）和整数索引 → 通常返回视图；花式索引（整数数组/列表下标）和布尔索引 → 总是返回副本。要独立拷贝用 .copy()。
Q8简答中等

什么是过拟合与欠拟合？为什么不能只根据训练误差选择模型？

参考答案
欠拟合：模型太简单，训练误差和泛化误差都高。过拟合：模型太复杂，训练误差很低但泛化误差高（测试集差）。
随复杂度↑，训练误差持续↓但泛化误差先↓后↑。只看训练误差会选到过拟合的复杂模型，泛化差；应用验证集/交叉验证评估泛化能力来选模型。
Q9简答中等

解释逻辑推理系统的"正确性（soundness）"与"完备性（completeness）"，并说明 ⊨ 与 ⊢ 的区别。

参考答案
⊨（语义蕴含）：满足 KB 的所有世界里 f 都为真；⊢（语法可证）：用推理规则能推出 f。
正确性：若 KB⊢f 则 KB⊨f（推出来的都真，不乱推）。完备性：若 KB⊨f 则 KB⊢f（所有真的都能推出）。
系统可"正确但不完备"（规则太少）；关键转化：KB⊨f ⟺ KB∪{¬f} 不可满足。
Q10简答基础

global 与 nonlocal 的作用与适用场景分别是什么？

参考答案
global x：在函数内修改全局变量绑定（如改模块级计数器）。
nonlocal x：在嵌套函数内修改最近的外层函数（非全局）的变量绑定（如闭包更新状态）。
背景：函数内只要对某名字赋值，默认就当它是局部变量；要改外层/全局须显式声明，否则会新建局部变量甚至触发 UnboundLocalError。

Q24实现中等

（真题原题）在一个二维有序列表（每行内升序、行间也升序，即把它看成展平后整体升序）上实现二分查找，找到目标返回 (行, 列)，找不到返回 (-1,-1)。

参考答案
思路：把 m×n 矩阵当成长度 m·n 的有序一维序列做二分，下标 mid 映射回 (mid//n, mid%n)。 
def search2d(mat, target):
    if not mat or not mat[0]:
        return (-1, -1)
    m, n = len(mat), len(mat[0])
    lo, hi = 0, m * n - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        v = mat[mid // n][mid % n]
        if v == target:
            return (mid // n, mid % n)
        elif v < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return (-1, -1)
易错：while lo<=hi（等号不能丢）；映射 mid//n, mid%n 用列数 n。
Q25实现中等

（真题原题）实现游戏角色基类 Character（含名字、血量、攻击力，方法：攻击、是否存活）和子类 Mage（法师，多一个法力值，新增释放法术技能，伤害翻倍）。

参考答案
class Character:
    def __init__(self, name, hp, atk):
        self.name, self.hp, self.atk = name, hp, atk
    def attack(self, other):
        other.hp -= self.atk
        return f"{self.name}攻击{other.name}，伤害{self.atk}"
    def is_alive(self):
        return self.hp > 0

class Mage(Character):
    def __init__(self, name, hp, atk, mp):
        super().__init__(name, hp, atk)   # 复用父类初始化
        self.mp = mp
    def cast(self, other):
        if self.mp < 10:
            return f"{self.name}法力不足"
        self.mp -= 10
        other.hp -= self.atk * 2
        return f"{self.name}释放法术，伤害{self.atk*2}"
给分点：基类放共有属性/方法；子类 super().__init__ 后再加自己的；新增/重写方法体现继承与多态。
Q26手写中等

设计二维向量类 Vector，支持加法、数乘、点乘和 str 表示，使 print(Vector(3,4)+Vector(1,2)) 得 (4, 6)，print(Vector(3,4)*Vector(1,2)) 得 11。

参考答案
class Vector:
    def __init__(self, x, y):
        self.x, self.y = x, y
    def __add__(self, o):
        return Vector(self.x+o.x, self.y+o.y)
    def __mul__(self, o):
        if isinstance(o, Vector):
            return self.x*o.x + self.y*o.y     # 点乘→标量
        return Vector(self.x*o, self.y*o)       # 数乘
    def __str__(self):
        return f"({self.x}, {self.y})"
要点：用 isinstance 区分点乘/数乘；需要 5*v 也支持则加 __rmul__。
Q27手写难

用 NumPy 手写 PCA：输入数据矩阵 X（m×D，按行存样本）和目标维度 d，返回降维后的 Z（m×d）。不许调用现成 PCA。

参考答案
import numpy as np
def pca(X, d):
    X = X - X.mean(axis=0)            # 1.中心化
    C = (X.T @ X) / X.shape[0]        # 2.协方差矩阵
    vals, vecs = np.linalg.eigh(C)       # 3.特征分解(对称用eigh)
    order = np.argsort(vals)[::-1]      # 4.特征值降序
    W = vecs[:, order[:d]]               # 5.取前d个特征向量
    return X @ W                        # 6.投影
易错：必须先中心化；argsort 默认升序要 [::-1] 反转；用 eigh 比 eig 稳。
Q28补全中等

补全 KNN 分类核心：给定到各训练点的距离 dist 和标签 y，取最近 k 个投票，平票选编号最小的类。

参考答案
import numpy as np
from collections import Counter
def vote(dist, y, k):
    idx = np.argsort(dist)[:k]            # 最近 k 个的下标
    labels = [y[i] for i in idx]
    cnt = Counter(labels)
    mx = max(cnt.values())
    return min(c for c, n in cnt.items() if n == mx)  # 平票取最小编号
要点：argsort 稳定，距离相同先出现的排前；平票用 min 选编号最小类。

Q31综合难

（真题原题）给定某棋类游戏规则，要在对局中预测双方胜率。请把该问题转成人工智能程序可处理的形式，说明大致思路，并说明效果不好如何改进。

参考答案

Q33算法分析中等

给定 5 条事务：T1{牛奶,面包}、T2{牛奶,尿布,啤酒}、T3{面包,尿布,啤酒}、T4{牛奶,面包,尿布,啤酒}、T5{牛奶,面包,尿布}。计算：①支持度 support({尿布,啤酒})；②置信度 confidence(啤酒→尿布)、confidence(尿布→啤酒)。并说明 Apriori 剪枝原理。

参考答案
含{尿布,啤酒}的事务：T2,T3,T4 → σ=3，support = 3/5 = 0.6。
σ(啤酒)=T2,T3,T4=3；σ(尿布)=T2,T3,T4,T5=4。
confidence(啤酒→尿布) = 3/3 = 1.0；confidence(尿布→啤酒) = 3/4 = 0.75。注意同一项集不同方向置信度不同。
Apriori 原理：频繁项集的所有子集必频繁 ⟺ 非频繁项集的所有超集必非频繁——据此剪掉含非频繁子集的候选，不再生成检验。
