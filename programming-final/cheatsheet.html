② 基础算法骨架

二分查找

def bsearch(a, target):
    lo, hi = 0, len(a) - 1
    while lo <= hi:               # 等号不能丢
        mid = (lo + hi) // 2
        if a[mid] == target: return mid
        elif a[mid] < target: lo = mid + 1
        else:               hi = mid - 1
    return -1
# 二维矩阵当展平一维：mat[mid//n][mid%n]

质数判定

def is_prime(x):
    if x < 2: return False
    for i in range(2, int(x**0.5) + 1):   # 只需到根号 x
        if x % i == 0: return False
    return True

def make_counter(s, d):
    t = -1

高阶：通用框架 + 复合 + 柯里化

def summation(n, term):              # 把"第k项"抽成参数
    return sum(term(k) for k in range(1, n+1))

def compose(f, g):                    # h(x)=f(g(x))
    return lambda x: f(g(x))

def curry2(f):                        # f(x,y) → f(x)(y)
    return lambda x: lambda y: f(x, y)

def repeat_apply(func):               # 连续作用 t 次
    def outer(t):
        g = lambda x: x
        for _ in range(t): g = compose(g, func)
        return g
    return outer

装饰器

def deco(fn):
    def wrapped(*args, **kw):
        # 前置增强
        r = fn(*args, **kw)
        # 后置增强
        return r
    return wrapped

class Rev:
    def __init__(self, lst):
        self.lst = list(reversed(lst)); self.i = 0
    def __iter__(self): return self
    def __next__(self):
        if self.i >= len(self.lst): raise StopIteration
        v = self.lst[self.i]; self.i += 1; return v

⑤ 机器学习骨架

PCA

def pca(X, d):
    X = X - X.mean(axis=0)
    C = (X.T @ X) / X.shape[0]
    vals, vecs = np.linalg.eigh(C)
    order = np.argsort(vals)[::-1]
    return X @ vecs[:, order[:d]]

KNN

def knn(train_X, train_y, x, k, p=2):
    d = [(sum(abs(a-b)**p for a,b in zip(x,v))**(1/p), j)
         for j,v in enumerate(train_X)]
    d.sort()                          # 距离同则下标小在前
    from collections import Counter
    cnt = Counter(train_y[j] for _,j in d[:k])
    mx = max(cnt.values())
    return min(c for c,n in cnt.items() if n==mx)  # 平票取最小

⑥ 逻辑推理模板

解释函数（递归求真值）

def interp(f, w):
    if isinstance(f, str): return w[f]
    op = f[0]
    if op=="not": return not interp(f[1], w)
    if op=="and": return interp(f[1],w) and interp(f[2],w)
    if op=="or":  return interp(f[1],w) or  interp(f[2],w)
    if op=="imp": return (not interp(f[1],w)) or interp(f[2],w)

前向链接（Horn 子句）

def forward(facts, rules):           # rules: [(前提集, 结论)]
    facts = set(facts); changed = True
    while changed:
        changed = False
        for prem, conc in rules:
            if prem <= facts and conc not in facts:
                facts.add(conc); changed = True
    return facts

易错点清单

人工智能程序设计 · 语法陷阱 / 边界条件 / 常踩的坑（考前快速扫一遍）

语法陷阱边界条件常踩的坑

① Python 基础

语法陷阱地板除负数向下取整-1 // 5 == -1（不是 0），7 // 2 == 3；/ 结果永远是 float（6/3==2.0）。
常踩的坑input 返回字符串input() 永远是 str，参与数值运算前必须 int()/float()/eval()。
常踩的坑强类型不隐式转换"5" + 5 报 TypeError；变量却可动态改类型（x=3; x="a" 合法）。
边界条件不可变对象"修改"换 idx=x+1 新建对象（id 变）；y=x 后改 x 不影响 y。
语法陷阱复数 j1j**2 == (-1+0j)；单写 j 是 NameError。

② 数据结构

常踩的坑a += [x] 是原地修改对 list 等价 extend，所有别名一起变；a = a + [x] 才新建。
常踩的坑原地方法返回 Nonesort()/reverse()/append() 返回 None；要新列表用 sorted()/reversed()。
常踩的坑浅拷贝不拷内层b=a.copy() 改 b[2][0] 会连带改 a 的嵌套子列表；要彻底独立用 deepcopy。
边界条件切片左闭右开a[:-1] 去掉最后一个；切片返回新列表不改原；越界索引才报 IndexError，切片越界不报。
语法陷阱不可变容器含可变元素t=([1,2],3)：t[0].append(9) 合法，t[0]=[..] 报错。
语法陷阱单元素元组 / 空集合(1,) 逗号不能少；空集合是 set()，{} 是空字典。
边界条件find vs indexfind 找不到返回 -1，index 找不到抛 ValueError；字典 d[k] 缺键 KeyError，d.get(k,默认) 安全。

③ 流程控制

常踩的坑短路返回操作数本身3 and 5 == 5，0 or "x" == "x"——不一定是 True/False。假值：0 None '' [] {} ()。
边界条件迭代器/生成器只用一次耗尽后再遍历为空；生成器表达式用圆括号且是生成器不是元组。
语法陷阱print 的 sep/endprint(a,b,sep="-",end="!")；默认 sep 空格、end 换行。

④ 函数与函数式

常踩的坑可变默认参数def f(val,lst=[]) 多次调用共享同一 list、累积；改用 lst=None 哨兵。
语法陷阱赋值即局部 / UnboundLocalError函数内只要给某名字赋值，它整段就是局部变量；想改外层/全局须 nonlocal/global。
常踩的坑闭包绑定循环变量循环里造多个函数，闭包共享同一变量、最终都拿末值；用默认参数固定：def g(y,a=x): ...。
边界条件map/filter 惰性返回迭代器，需 list() 才看到值，且只能消费一次。
边界条件递归终止条件必写基准情形，否则无限递归（深度约 1000）；重复子问题用字典记忆化防超时。
语法陷阱参数定义顺序位置 → 默认 → *args → 仅关键字 → **kwargs；调用时位置参数必在关键字前。
边界条件多返回值是元组return a,b 返回 (a,b)，可整体接或解包；无 return 返回 None。

⑤ 面向对象

常踩的坑类属性遮蔽obj.x=v 总在实例上新建、遮蔽类属性；改类属性须用 类.x=v。
常踩的坑子类重写 __init__ 漏 super子类 __init__ 不调 super().__init__(...)，父类属性全丢。
边界条件super 是 MRO 下一个多继承里 super() 不是"父类"而是 MRO 里下一个；"先 super 再打印"的输出顺序与 MRO 相反。
常踩的坑Duck Typing 运行时才报错调用前不查类型，缺方法时运行到才抛 AttributeError；可用 hasattr/try-except 防。
语法陷阱name mangling__x 实际改名 _类名__x，外部 obj.__x 访问失败；非加密。
边界条件混合类型运算运算符重载遇不支持类型应返回 NotImplemented，触发反向方法 __radd__ 等。

⑥ 异常 / 模块

常踩的坑finally 的 return 覆盖finally 里 return 会盖掉 try/except 的返回值。
边界条件except 匹配顺序先匹配先处理；except Exception 写最前会吞掉一切，具体异常要写前面。
常踩的坑__exit__ 返回值决定异常去留返回 True 压制异常、None/False 继续抛；with 本身不等于捕获异常。
语法陷阱自定义异常漏 super构造里必须 super().__init__(msg)，否则消息为空。
常踩的坑缺 __name__ 守卫无 if __name__=="__main__": 时，模块被 import 也会跑顶层测试代码。

⑦ NumPy

边界条件视图 vs 副本切片/整数索引是视图（改连带改原），花式/布尔索引是副本（不影响原）。
常踩的坑广播从右对齐维度不足左补 1；每维相等或有一个为 1 才兼容，否则报错。判 shape 别从左对。
边界条件axis 方向axis=0 压行得每列、axis=1 压列得每行；聚合后该轴消失。
常踩的坑重复索引累加a[idx]+=1 重复索引只加一次；要逐次累加用 np.add.at。
常踩的坑转置后 reshape 可能复制.T 只改 strides 不动数据；其后默认 reshape 因内存不连续会触发复制。
边界条件where 两种用法np.where(c,x,y) 返回数组；np.where(c) 返回索引元组。除零警告用 np.errstate 屏蔽。

⑧ SymPy / SciPy / Pandas

边界条件SymPy Float 精度Float(0.3,25) 不精确；要精确传字符串 Float('0.3',25)。Sum/Product 要 .doit()。
常踩的坑linprog 默认最小化最大化要目标系数取反、结果再取反；>= 约束两边乘 -1；bounds 默认 (0,None)。
常踩的坑quad 被积函数要向量化用 np.exp 不用 math.exp；返回 (值,误差) 二元组要解包。solve 要非奇异方阵。
常踩的坑Pandas 索引对齐生 NaN不同 index 的 Series/DataFrame 运算只对齐共同标签，单方独有为 NaN、类型升 float64；用 fill_value/fillna 处理。
边界条件merge 默认 inner不匹配的键被丢；键非一一对应会产生笛卡尔积式扩张。

⑨ 机器学习

边界条件PCA 必须中心化不中心化第一主成分会偏向均值方向；特征尺度差异大要先 z-score；特征向量符号 v/-v 都对（作业要按规则定符号）。
常踩的坑规范化参数只从训练集算min/max、μ/σ 必须训练集拟合、测试集套用，防数据泄露；常数列 σ=0 要防除零。
边界条件KNN 平票 / 距离同序平票取类别编号最小；距离相同训练集先出现的排前（排序键带下标保稳定）。k 大欠拟合、k 小过拟合。
常踩的坑K-Means 初始敏感可能收敛局部最优；作业里初始中心常固定取前 k 个点；K-Centroid 中心必须是真实样本点（非均值）。
语法陷阱置信度用整数比较判 100*σ(X∪Y) >= C*σ(X) 避免浮点误差。
常踩的坑逻辑回归是分类名字带"回归"但做分类；sigmoid 分段写防溢出；阈值 0.5 决策。

⑩ 逻辑推理

边界条件蕴含真值f→g 只有前真后假才为假；前件为假时整体恒真。
常踩的坑无效推理形式肯定后件（Q,P→Q ⇒ P）、否定前件（¬P,P→Q ⇒ ¬Q）都无效。
边界条件⊨ vs ⊢KB⊨f（语义）⟺ KB∪{¬f} 不可满足；⊢ 是语法可证。正确≠完备。
常踩的坑Apriori 方向"非频繁项集的超集必非频繁"用于剪枝；不能反过来由超集推子集。
