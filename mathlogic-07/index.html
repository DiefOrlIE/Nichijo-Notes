为什么要"语义"？——从无意义的符号串到真假

补充背景 · 进入正文前必看

上一章你已经能造出像下面这样的"合法字符串"：

∀x. <(x, S(x))

但请问：这句话是真是假？ ——答不出来。因为到目前为止 ∀、<、S、x 都只是字符；连 x 取值范围是什么我们都没说。

这一章的任务

给上一章造的纯语法对象（项、公式）一个 解释（interpretation），即指定：

  • 论域（变元跑在哪个集合上，是 ℕ 还是 ℝ 还是某个群？）
  • 非逻辑符号的含义（S 是后继函数？< 是小于？常元 0 是哪个元素？）
  • 变元的当前取值（自由变元像 x₁ 现在取多少？）

给定这一切后，一个公式就有明确的真假可言了。

类比命题逻辑：命题逻辑里一句 p ∧ q，必须先给 p, q 各赋一个真假值才能算 p ∧ q 的真假。一阶逻辑也是同样的需求——只是现在原子不再是裸命题变元，而是 P(t₁,…,tₙ) 和 t₁ ≐ t₂，所以"赋真值"前还要先把 tᵢ 算成论域里的元素。

本章一句话：把上一章的符号串"翻译"成有意义的数学陈述，定义它们对一个具体数学对象（结构+赋值）的真假。这就是模型论的起点。

命题逻辑 vs 一阶逻辑 · 语义对照表

补充背景

你的命题逻辑直觉大多数都能迁移过来，只是更复杂一点：

命题逻辑一阶逻辑

原子怎么算v 直接查表P(t₁,…,tₙ)：先算 tᵢ ∈ M，再查 P_M ⊆ Mⁿ

三个"等"的预告（最容易混淆的字符）

补充背景

记号名字层级含义

结构 𝕄 = (M, I) ── 给非逻辑符号配"实物"

PDF 第 2–10 页（累积幻灯片，仅讲终态 p10）· 当前在 第 ① 层

核心定义（定义 3.10）

固定一阶语言 ℒ。ℒ 的结构 𝕄 = 二元组 (M, I)：

  • M（论域 / domain）：非空集合，变元在这里取值。白话：你在哪个世界里讨论问题。
  • I（解释 / interpretation）：把 ℒ 的非逻辑符号映射到这个世界里的实物。

I 的四条要求（按符号种类分）

符号种类I 把它送到哪白话
(a) 常元 cI(c) ∈ M常元 → 论域里的一个固定元素
(b) n 元函数符 fI(f) : Mⁿ → M函数符 → 真函数
(c) 0 元谓词 PI(P) ∈ 𝐁={T,F}0 元谓词 = 命题，给个真值
(d) n 元谓词 P (n≥1)I(P) ⊆ Mⁿn 元谓词 → 一个 n 元关系

具体例子：初等算术语言 𝒜 的标准模型 ℕ

𝒜 的非逻辑符号：常元 {0}，函数符 {S, +, ·}（S 是 1 元，+ 和 · 是 2 元），谓词符 {<}（2 元）。

取 N = {0,1,2,…}（自然数集），定义解释 I：

I(0)=0，I(S)=suc（后继函数），I(+)=+（自然数加法），I(·)=×（乘法），I(<)={⟨0,1⟩,⟨0,2⟩,⟨1,2⟩,…}（小于关系）

这样得到的 ℕ = (N, I) 称为初等算术的标准模型。

关键直观：谓词被解释成关系（即满足关系的有序组的集合）。比如要表达"< 在 ℕ 上"，就是列出所有 (a,b) 满足 a<b 的组成的集合。

群论语言 𝔊 的例子（对照）

𝔊 = ({e}, {·, ⁻¹}, ∅)（一个常元 e，二元运算 ·，一元运算 ⁻¹，没有谓词）。一个 𝔊-结构就是一个具体的群，比如整数加群 (ℤ, +, 0, −·)：M=ℤ，I(e)=0，I(·)=+，I(⁻¹)=负号。同一个语言可以有无穷多种结构。

"语言 ℒ 给出可写的句子模板，结构 𝕄 给出这些模板说的是哪个具体的数学对象"——这是语法/语义分离的精髓。

记号约定 & "M 代表 𝕄"

PDF 第 11 页

下标约定

为了书写方便，I(c)、I(f)、I(P) 分别简记为 c_M、f_M、P_M。下标 M 提示这是"在结构 𝕄 中的解释"。

cM := I(c),   fM := I(f),   PM := I(P)

I 的总像

把所有解释合起来看，I 的取值范围是：

I : ℒ → M ∪ F ∪ 𝐁 ∪ 𝓜

其中 F 表 M 上各阶函数的集合，𝓜 表 M 上各阶关系的集合。这一行只是把上面四条要求合并写。

滥用记号：M 代表 𝕄

书写习惯：以后写"结构 M"和"论域 M"是同一个字母，因为论域决定了大半个结构。比如"结构 ℕ"和"自然数集 ℕ"在上下文里通常不区分——只要心里知道结构其实是 (N, I) 的二元组就行。

项 vs 公式 ── 两层归纳，次序不能颠倒

补充背景 · 进入项/公式解释前必看

下一节起会同时定义两种"解释"。请先把它们的类型分清：

项的解释公式的解释
记号tM[σ]AM[σ]
取值集合∈ M（论域元素）∈ 𝐁 = {T, F}
归纳基础变元、常元原子公式 P(…)、t₁≐t₂
归纳步骤函数项 f(t₁,…,tₙ)¬, ∧/∨/→, ∀x, ∃x
为什么必须先项后公式？原子公式形如 P(t₁,…,tₙ) ——它的真假要看 (t₁M[σ],…,tₙM[σ]) ∈ P_M。所以你必须先把里面每个项算成论域元素，才能拿去查关系 P_M。次序错了就无意义。

形象比喻

把 tM[σ] 想成"表达式求值器"，把 AM[σ] 想成"真值判定器"。求值器吐出来的是数（M 的元素），判定器吐出来的是真假。第二个调用第一个。

赋值 σ 与模型 (𝕄, σ)

PDF 第 12–14 页（累积，讲终态）· 当前在 第 ② 层

定义 3.11

  • 赋值 σ：从变元集 V={x₀,x₁,…} 到论域 M 的函数 σ: V→M。
    白话：给每个变元符指一个论域里的具体值。
  • 模型：二元组 (𝕄, σ)，结构 + 赋值，也写成 (M, σ)。
为什么结构里没有把变元解释了？因为变元的值要可以变（量词遍历时需要"换值"），所以单独抽出 σ。结构 𝕄 是"固定的舞台"，赋值 σ 是"演员当前的位置"。

𝒜 标准模型上的一个常用赋值

对 ℕ = (N, I)，取 σ(xₙ) = n。这个 σ 在后面大量例题中默认使用——记法："令 σ(xₙ)=n"。这样 x₀↦0、x₁↦1、x₇↦7…。

σ[x:=a] 的几何直观 ── 钉一个变元，其他不变

补充背景 · 进入定义 3.12 前必看

下一页会引入一个超级核心的记号 σ[x:=a]。先看一个图：

把 σ 想成一张"变元-值"对照表：

σ: x₀↦0, x₁↦1, x₂↦2, x₃↦3, x₄↦4, x₅↦5, …

那么 σ[x₃ := 99] 就是：

σ[x₃:=99]: x₀↦0, x₁↦1, x₂↦2, x₃↦99, x₄↦4, x₅↦5, …

只把第 3 行覆盖成 99，其他都不动。

为什么这个记号那么重要？量词 ∀x.A 的语义是"对所有 a∈M，把 x 暂时设为 a 后 A 真"。这个"暂时设为 a"正是 σ[x:=a] —— 别处的变元 (像 x₁, x₂ 中的自由变元) 都不能动，否则就不是"换 x"而是"换整个赋值"了。
易错点：σ[x:=a] 不是"把 σ 删了重做"，而是"打补丁"。下面所有量词处理都依赖这个区别。

定义 3.12：σ[x:=a] 的精确写法

PDF 第 15 页

形式定义：

(σ[xi := a])(xj) = { σ(xj) 当 i≠j；  a 当 i=j }

就是分两种情况：问的变元下标和被覆盖的不一样→原样返回；一样→返回新值 a。完全对应上一页的图。

定义 3.13：项的解释 tM[σ]

PDF 第 16–17 页（累积，讲终态）· 当前在 第 ③ 层（项侧）

归纳定义（按项的结构）

项的形状解释白话
变元 xxM[σ] = σ(x)变元的值=查赋值
常元 ccM[σ] = cM常元的值=结构里固定的实物（不依赖 σ）
函数项 f(t₁,…,tₙ)= fM((t₁)M[σ],…,(tₙ)M[σ])先把每个子项算成元素，再用 f_M 这个真函数去算
想象一个"求值树"——叶子是变元/常元（直接出值），内部节点是 f，递归往下算每个子树，再用 f_M 把孩子的值算合。这就是项归纳。

下一页用 𝒜 标准模型走一个具体例子，你看一遍就清楚了。

项解释例：1 + suc(7) = 9

PDF 第 18–19 页（累积，讲终态）

题目

在 𝒜 的标准模型 (N, σ)，其中 σ(xₙ)=n，求项 +(x₁, S(x₇)) 的解释。

逐步展开（项归纳）

(+(x₁, S(x₇)))_N[σ]
  = +_N( (x₁)_N[σ], (S(x₇))_N[σ] )         # 规则 3：拆函数项
  = σ(x₁) + (S(x₇))_N[σ]                    # x₁ 用规则 1, +_N=+
  = 1     + suc((x₇)_N[σ])                   # S(x₇) 拆开, S_N=suc
  = 1     + suc(σ(x₇))                       # x₇ 用规则 1
  = 1     + suc(7)
  = 1 + 8
  = 9
每一步只动了"一个节点"。这种机械化展开是结构归纳的常见做法。等到熟练后可以一步到位写"=1+8=9"。

引理 3.14：项的解释一定落在 M 里

PDF 第 20–21 页（累积，讲终态）

引理

对任意项 t、任意模型 (𝕄, σ)：tM[σ] ∈ M。

证明（对项结构归纳）

  • 基础：t 是变元 x 时，tM[σ] = σ(x)，由 σ:V→M 立得 ∈ M。t 是常元 c 时，tM[σ] = cM = I(c) ∈ M（定义 3.10(a)）。
  • 归纳：t = f(t₁,…,tₙ) 时，由归纳假设 (tᵢ)M[σ] ∈ M。由定义 3.10(b) 知 fM:Mⁿ→M，所以代入得 tM[σ] = fM(…) ∈ M。
这条引理看起来废话，但是非常关键：它保证了下一节谓词原子 P(t₁,…,tₙ) 的真假定义不会出问题——你要查的 (t₁M[σ],…) 真的在 Mⁿ 里。

右边累积框 (1)–(5) 给出了 𝒜 标准模型上的具体计算结果，比如 (S(xᵢ))_N[σ] = σ(xᵢ)+1。和上一页的例子是一回事。

命题（0 元谓词）的解释

PDF 第 22 页 · 当前在 第 ③ 层（公式侧）

结构定义里已经规定：0 元谓词 P 的解释 I(P) ∈ 𝐁 = {T,F}。所以一个 0 元谓词在任何模型下直接就是 T 或 F——和命题逻辑里"原子命题"的角色完全一样。

这是从命题逻辑过渡到一阶逻辑最自然的桥梁：命题逻辑的原子全部对应一阶逻辑的 0 元谓词。

联结词解释 (1)：¬ 和 ∧

PDF 第 23 页

把联结词解释成 𝐁 上的真值函数 B¬, B∧, B∨, B→，跟命题逻辑完全相同。这里只是换了符号：

B¬: 𝐁→𝐁，B∧,B∨,B→: 𝐁×𝐁→𝐁

真值表你已经背熟了，跳过即可。

联结词解释 (2)：∨ 和 → · "与命题逻辑一致"

PDF 第 24–25 页（累积，讲终态）

定义 3.15：公式的解释 AM[σ] ── 全章核心定义

PDF 第 26–27 页（累积，讲终态）

对公式 A 按结构归纳定义 AM[σ] ∈ 𝐁。六条规则：

公式形状解释白话
(1) P(t₁,…,tₙ)T 当 ⟨(t₁)M[σ],…⟩ ∈ P_M；否则 F先把项算出来，再查 P_M 这张关系表里有没有这个组
(2) t₁ ≐ t₂T 当 (t₁)M[σ] = (t₂)M[σ]；否则 F等词 ≐：算出来在 M 里相等就真
(3) ¬AB¬(AM[σ])查 ¬ 的真值表
(4) A * BB*(AM[σ], BM[σ])* 是 ∧/∨/→ 任一，查二元真值表
(5) ∀x.AT 当 ∀a∈M, AM[σ[x:=a]] = T；否则 F遍历整个论域，每个 a 都使内部真才整体真
(6) ∃x.AT 当 ∃a∈M, AM[σ[x:=a]] = T；否则 F找到一个 a 让内部真即可
易错：等词 ≐ 和元语言相等 = 不要混。(2) 里左边的 ≐ 是对象语言的二元谓词；右边的 = 是元语言里"两个论域元素是同一个"——一个是公式里的字符，一个是数学陈述。看到 ≐ 渲染异常请用 Chrome。

(5) (6) 是全章最核心的两条。注意"遍历论域 M"——若论域无穷（如 ℕ），你不能用真值表暴力枚举。所以判定一阶公式真假是不可判定的（哥德尔后续会告诉你）。但在具体证明里，我们通过取任意 a 或构造特定 a 来推理。

下面一页就拿这条规则跑一个完整例子。

公式解释例：∀x₃.(x₃ < x₁ + x₄) 在标准模型上的求值

PDF 第 28–33 页（累积 6 张，讲终态）

题目

𝒜 标准模型 ℕ, σ(xₙ)=n。求 (∀x₃. (<(x₃, +(x₁, x₄))))N[σ]，可读作 ∀x₃.(x₃ < x₁+x₄)。

分两层展开

外层 ∀x₃（用规则 (5)）：值为 T 当且仅当 对所有 a ∈ N，把 σ 在 x₃ 处覆盖成 a 后，内部公式 (x₃ < x₁+x₄)N[σ[x₃:=a]] = T。

内层 x₃ < x₁ + x₄（谓词 < 用规则 (1)）：值为 T 当且仅当 ⟨(x₃)N[σ[x₃:=a]], (+(x₁,x₄))N[σ[x₃:=a]]⟩ ∈ <N。

把每个项算出来

(x₃)_N[σ[x₃:=a]]      = (σ[x₃:=a])(x₃) = a       # 关键：被覆盖成 a
(x₁)_N[σ[x₃:=a]]      = (σ[x₃:=a])(x₁) = σ(x₁) = 1   # x₁ 没被覆盖
(x₄)_N[σ[x₃:=a]]      = (σ[x₃:=a])(x₄) = σ(x₄) = 4   # x₄ 没被覆盖
(+(x₁,x₄))_N[σ[x₃:=a]] = 1 + 4 = 5

所以内层等价于：⟨a, 5⟩ ∈ <_N，即 a < 5。

整体结果

∀a ∈ N，a < 5？ —— 显然不成立（取 a=5,6,7,… 反例无穷多）。所以整体值为 F。
关键体感：σ[x₃:=a] 只钉了 x₃，x₁, x₄ 还是用旧的 σ 值（1 和 4），所以"+ (x₁,x₄)"算出来是常数 5 与 a 无关。然后量词遍历 a，问"a < 5 是否对所有自然数成立"——很容易看出 F。
易混：如果题目是 ∃x₃.(x₃ < x₁+x₄)，答案就是 T（取 a=0 即可）。同样的赋值，量词不同，答案完全相反。

引理 3.16'：公式解释一定是真假之一

PDF 第 34 页

对任何公式 A，AM[σ] ∈ 𝐁 = {T,F}。证明思路：对公式结构作归纳，每一条规则要么直接给出 T/F，要么经 B¬,B* 这些真值函数（值域 ⊆ 𝐁）。

这条和引理 3.14（项落 M 里）配套——保证整个语义系统是"良定义的"。

定义 3.15'：用 ⊨σ 重写的等价定义

本质就是把定义 3.15 用 M ⊨σ A 代替 AM[σ] = T 重新写一遍，更接近逻辑文献的习惯写法。

公式⊨σ 形式
t₁ ≐ t₂iff (t₁)M[σ] = (t₂)M[σ]

引理 3.16：公式真值只看自由变元

PDF 第 36 页

引理

设 (M, σ₁), (M, σ₂) 是 ℒ-模型。若 σ₁ 和 σ₂ 在 FV(A) 上相同（记 σ₁|FV(A) = σ₂|FV(A)），则 AM[σ₁] = AM[σ₂]。

白话：给公式 A 求真假时，只用得到 A 里自由出现的那些变元的值。约束变元值改了不影响——因为约束变元的值是被量词"局部覆盖"的，外部赋值给它什么没意义。

书上的小例

(∀x. x ≐ y)M[σ[z:=a]] = (∀x. x ≐ y)M[σ]。
因为 FV(∀x. x≐y) = {y}，里面没有 z；所以把 σ 在 z 处改成 a 完全不影响这个公式的值。

为什么这条引理重要：它把"公式的真假"从依赖整个 σ 退化到"只依赖 FV(A) 上的 σ"。对闭公式（FV(A)=∅），值完全由 M 决定，与 σ 无关——这就是后面 M⊨A 的语义基础。

接下来 5 页 (p38–p42) 是完整的归纳证明。这是本章最详细的结构归纳示范，强烈建议跟着走一遍。

证明 (1/5)：先对项归纳证 (*)

PDF 第 37–38 页（p37 与 p38 重复，仅讲 p38）

分两层归纳

第一步先证明项版本（记为 (*)）：

若 σ₁|FV(t) = σ₂|FV(t)，则 tM[σ₁] = tM[σ₂]

项归纳

  • 基础：t 为变元 x 时，FV(t)={x}，所以 σ₁(x)=σ₂(x)，故 tM[σ₁]=σ₁(x)=σ₂(x)=tM[σ₂]。t 为常元时 FV(t)=∅，两边都等于 cM。
  • 归纳假设：对 t₁,…,tₙ 都有 (*)。
  • 归纳步骤：t = f(t₁,…,tₙ) 时，FV(f(t₁,…,tₙ)) = ⋃ FV(tᵢ)。条件 σ₁|FV(t)=σ₂|FV(t) 推出 σ₁|FV(tᵢ)=σ₂|FV(tᵢ)，由 IH 知 (tᵢ)M[σ₁]=(tᵢ)M[σ₂]，故：
    f(t₁,…,tₙ)_M[σ₁] = f_M((t₁)_M[σ₁],…,(tₙ)_M[σ₁])
                  = f_M((t₁)_M[σ₂],…,(tₙ)_M[σ₂])     # 用 IH
                  = f(t₁,…,tₙ)_M[σ₂]                  ✓
这就是结构归纳的标准三段套路：基础、IH、归纳步——你已经在第 6 章见过很多次。后面公式归纳完全是同一个模板。

证明 (2/5)：公式归纳 · 情况 1 等词

PDF 第 39 页

有了 (*)，再对公式 A 做结构归纳。归纳基础有两种原子公式：等词和谓词。

情况 1：A = (s ≐ t)

FV(A) = FV(s) ∪ FV(t)。前提 σ₁|FV(A)=σ₂|FV(A)，所以 σ₁|FV(s)=σ₂|FV(s) 且 σ₁|FV(t)=σ₂|FV(t)。由 (*) 得 sM[σ₁]=sM[σ₂]，tM[σ₁]=tM[σ₂]。则：

A_M[σ₁] = (s≐t)_M[σ₁]
        = { T 若 s_M[σ₁] = t_M[σ₁];  F 否则 }
        = { T 若 s_M[σ₂] = t_M[σ₂];  F 否则 }   # 项值相等
        = A_M[σ₂]                                ✓

证明 (3/5)：情况 2 谓词

PDF 第 40 页

情况 2：A = P(t₁,…,tₙ)

FV(A) = ⋃ FV(tᵢ)。同样推出 σ₁|FV(tᵢ)=σ₂|FV(tᵢ)，由 (*) 得 (tᵢ)M[σ₁]=(tᵢ)M[σ₂]。代入：

A_M[σ₁] = T iff ⟨(t₁)_M[σ₁],…⟩ ∈ P_M
        = T iff ⟨(t₁)_M[σ₂],…⟩ ∈ P_M   # 元组逐位相等
        = A_M[σ₂]                       ✓
情况 1, 2 都依赖项侧的 (*)，这就是为什么必须先证项版本再证公式版本——双层归纳的次序不能换。

证明 (4/5)：情况 3, 4 联结词

PDF 第 41 页

情况 3：A = ¬B

FV(¬B) = FV(B)。由 σ₁|FV(B)=σ₂|FV(B) 和 IH 得 BM[σ₁]=BM[σ₂]。代入：

A_M[σ₁] = B_¬(B_M[σ₁])
        = B_¬(B_M[σ₂])     # IH
        = A_M[σ₂]            ✓

情况 4：A = B * C（*∈{∧,∨,→}）

书上写"略"。同理：FV(B*C)=FV(B)∪FV(C)，对 B, C 分别用 IH 得到值相等，再用真值函数 B_*。和情况 3 完全平行。

证明 (5/5)：情况 5, 6 量词 ── 最关键

PDF 第 42 页

情况 5：A = ∀x.B

FV(∀x.B) = FV(B) − {x}。前提 σ₁|FV(A) = σ₂|FV(A)。关键一步：对任意 a∈M，

σ₁[x:=a] | FV(B) = σ₂[x:=a] | FV(B)

为什么？对 x：两边都是 a，相等。对 FV(B)−{x} 中的 y：σ₁(y)=σ₂(y)（来自前提），σ[x:=a] 不动 y，相等。

所以可以对内部 B 用 IH，得 BM[σ₁[x:=a]] = BM[σ₂[x:=a]]。再用 ∀ 的语义：

A_M[σ₁] = T iff ∀a∈M, B_M[σ₁[x:=a]] = T
        = T iff ∀a∈M, B_M[σ₂[x:=a]] = T   # IH 对每个 a
        = A_M[σ₂]                            ✓

情况 6：A = ∃x.B —— 同理

这一步的核心技巧：要对量词内部用归纳假设，必须先证明"打补丁后的赋值在 FV(B) 上仍然相同"。这是处理量词归纳的标准套路，第 8 章证证明系统的可靠性也会重复用到。

∀ 与 ∃ 的语义对偶 ── 小结一张表

补充背景 · 进入可满足前的小结

到这里量词的语义你已经完全清楚，但请把对偶性记在脑子里：

记号真当且仅当假当且仅当反例 / 见证
∀x.A所有 a∈M, A[x:=a] 真存在某个 a∈M, A[x:=a] 假找一个反例 a₀
∃x.A存在某个 a∈M, A[x:=a] 真所有 a∈M, A[x:=a] 假找一个见证 a₀
¬∀x.A ≃ ∃x.¬A       ¬∃x.A ≃ ∀x.¬A    （德摩根的量词版）
证题模板：要证 ∀x.A 真→任取 a∈M，证 A[x:=a] 真；要证 ∀x.A 假→构造一个 a₀ 使 A[x:=a₀] 假；要证 ∃x.A 真→构造见证 a₀；要证 ∃x.A 假→任取 a，证 A[x:=a] 假。这套模板贯穿后面所有证题。

第 1 个等价 ¬∀x.A ≃ ∃x.¬A 就是 p61 的标准例题。

定义 3.17：可满足 ── 六种含义

PDF 第 43–45 页（累积，讲终态）· 当前在 第 ④ 层

"可满足"四个字根据上下文有六种不同含义，PDF 一次列全：

说法记号含义
(1) 公式 A 对 (M,σ) 可满足M⊨σAAM[σ] = T
(2) 公式 A 可满足—存在 (M,σ) 使 M⊨σA
(3) M 满足 AM⊨A对所有 σ 都有 M⊨σA
(4) 公式集 Γ 对 (M,σ) 可满足M⊨σΓ∀A∈Γ, M⊨σA
(5) 公式集 Γ 可满足—存在 (M,σ) 使 M⊨σΓ
(6) M 满足 ΓM⊨Γ对所有 σ 都有 M⊨σΓ
易混点（warn）："A 可满足"≠"M⊨A"。前者只要找到一个模型就行（弱）；后者要求在这一个 M 下对所有赋值都真（强）。
举例：A = (x₀ ≐ x₁)。它"可满足"（取 σ(x₀)=σ(x₁)=0）但 ℕ⊭A（取 σ(x₀)=0, σ(x₁)=1 反例）。

右下角小框：M⊭σA 表示 AM[σ]=F；M⊭A 表示存在 σ 使 AM[σ]=F；M⊭σΓ 表示 ∃A∈Γ, AM[σ]=F；M⊭Γ 表示存在 σ 使 M⊭σΓ。否定记号的层级一定要数清。

定义 3.18：永真

PDF 第 46 页
  • A 永真，记 ⊨A：对所有模型 (M,σ) 都有 M⊨σA。
  • Γ 永真，记 ⊨Γ：对所有模型 (M,σ) 都有 M⊨σΓ。
"永真"是命题逻辑里"重言式"的一阶推广。⊨A 之所以左侧空白，正对应"无任何假设下 A 总真"。

定义 3.19：语义结论 Γ⊨A

PDF 第 47 页

Γ⊨A：对任何模型 (M,σ)，若 M⊨σΓ，则 M⊨σA。
白话：Γ 真的模型一定也让 A 真。

  • Γ⊭A：存在反例模型，使 Γ 真但 A 假。
  • ∅⊨A iff A 永真，即 ⊨A。空前提下的结论恰好是永真。
这是一阶逻辑里"逻辑推理"的语义版本——直接说"从 Γ 真推出 A 真"。下章会有它的语法版本（证明系统），并最终证可靠性 + 完备性。

三大形式逻辑基本定律

PDF 第 48 页

本节用反证法证三条永真公式，全是逻辑学起源时就被亚里士多德指出的"思维基本定律"：

名字形式古典动机
排中律 LEM⊨ A ∨ ¬A命题非真即假，没有第三种
矛盾律 LNC⊨ ¬(A ∧ ¬A)同时真又假不可能
同一律⊨ ∀x.(x ≐ x)任何东西等于自己
证法模板完全一样：反证 + 推出 AM[σ] = T 且 = F 的矛盾。

排中律：⊨ A ∨ ¬A

PDF 第 49–50 页（累积，讲终态）

反证

假设存在模型 (M,σ) 使 (A∨¬A)M[σ] = F。展开：

(A ∨ ¬A)_M[σ] = B_∨(A_M[σ], (¬A)_M[σ])
              = B_∨(A_M[σ], B_¬(A_M[σ])) = F   ← 假设

由 B_∨ 真值表，只有 F∨F=F，所以 AM[σ]=F 且 B_¬(AM[σ])=F。
后者由 B_¬ 真值表给出 AM[σ]=T。
于是 AM[σ] 同时 =T 又 =F，矛盾。

注意整个证明没用到 A 是什么，只用到 B_¬, B_∨ 的真值表——所以排中律对任意公式 A 成立，包括含变元、含量词的。

★ 矛盾律：⊨ ¬(A ∧ ¬A)（自补 · PDF 略）

PDF 原文略 · 本讲稿补全

反证

假设存在模型 (M,σ) 使 ¬(A∧¬A)M[σ] = F。展开：

(¬(A ∧ ¬A))_M[σ] = B_¬((A ∧ ¬A)_M[σ]) = F                              ← 假设
故 (A ∧ ¬A)_M[σ] = T                                                      # B_¬ 真值表
即 B_∧(A_M[σ], (¬A)_M[σ]) = T                                            # 展开 ∧
   B_∧(A_M[σ], B_¬(A_M[σ])) = T                                          # 展开 ¬A
由 B_∧ 真值表，T∧X=T 仅当 X=T，所以 A_M[σ] = T 且 B_¬(A_M[σ]) = T
后者由 B_¬ 真值表得 A_M[σ] = F
矛盾：A_M[σ] 同时 = T 且 = F。 ✓
和排中律的证明结构完全镜像。本质上排中律和矛盾律是德摩根对偶：A ∨ ¬A 和 ¬(¬A ∧ ¬¬A) ≃ ¬(¬A ∧ A) 在经典逻辑里互推。你可以验证它们的"反证树"互为镜像。
历史小知识：直觉主义逻辑放弃了排中律但保留矛盾律——这两条在经典逻辑下等价，但在更弱的逻辑系统下不一定等价。第 8 章会接触到这种细致区分。

同一律：⊨ ∀x.(x ≐ x)

PDF 第 51–52 页（累积，讲终态）

反证

假设存在模型 (M,σ) 使 (∀x.(x≐x))M[σ] = F。
由 ∀ 的语义：存在 a∈M，(x≐x)M[σ[x:=a]] = F。

但 (x≐x)M[σ[x:=a]] 要 = F，按 ≐ 的语义需要 xM[σ[x:=a]] ≠ xM[σ[x:=a]]——这是元语言的自己不等于自己，不可能。所以 (x≐x)M[σ[x:=a]] = T 对所有 a 成立，于是 (∀x.(x≐x))M[σ] = T，矛盾。

同一律本质就是"元语言里 a=a 永远成立"在对象语言里的体现。等词 ≐ 的语义是对照元语言的相等 =，所以左右两边写一样的项 x，必然真。

引理 3.20：Γ⊨A 等价于 Γ∪{¬A} 不可满足

PDF 第 53 页

反证：假设 Γ∪{¬A} 可满足，则存在 (M,σ) 使 M⊨σΓ 且 M⊨σ¬A。后者即 AM[σ]=F。但 Γ⊨A 要求 M⊨σΓ 蕴含 M⊨σA，矛盾。

这条引理是反证法的语义版——"要证 Γ⊨A，等价于证 Γ∪{¬A} 一定矛盾"。后面证明系统讨论"一致性"时是核心工具。

综合例：⊨ ∀x.(A→B) → (∀x.A → ∀x.B)

PDF 第 54–59 页（累积 6 张，讲终态）· 本章压轴例题

命题

证 ∀x.(A→B) ⊨ ∀x.A → ∀x.B。
白话：如果对所有 x A→B 真，且对所有 x A 真，那对所有 x B 也真。

反证 · 完整六步

假设 ∀x.(A→B) ⊭ ∀x.A→∀x.B
存在模型 (M,σ) 使：
  (1) (∀x.(A→B))_M[σ] = T                              # 假设前件真
  (2) (∀x.A → ∀x.B)_M[σ] = F                            # 假设后件假

由 (2) 与 → 真值表：
  (3) (∀x.A)_M[σ] = T 且 (∀x.B)_M[σ] = F               # → 唯一 F 的情况

展开 (3) 的两部分：
  (4) 对 ∀a ∈ M, A_M[σ[x:=a]] = T                       # ∀x.A=T
  (5) 存在 b ∈ M, B_M[σ[x:=b]] = F                       # ∀x.B=F

把 (4) 在 a=b 处实例化：
  (6) A_M[σ[x:=b]] = T 且 B_M[σ[x:=b]] = F

由 → 的真值表：
  (7) (A → B)_M[σ[x:=b]] = F                           # T → F = F

但 (1) 给出 ∀a∈M, (A→B)_M[σ[x:=a]] = T，取 a=b：
  (A → B)_M[σ[x:=b]] = T                                # 与 (7) 矛盾  ✓
关键技巧：(4)、(5) 都是"对所有 a / 存在 b"的陈述，要让它们对接，需要在同一个 b 上实例化。这是一阶语义题里最常用的招——把全称语句"实例化到见证"。
注意：这个结论的反方向 ∀x.A→∀x.B ⊨ ∀x.(A→B) 是不成立的！（思考反例：M=ℕ, A:=(x≐0), B:=(x≐0)；A→B 永真——但若换成 A:=(x≐0), B:=⊥ 就能造反例。）这种细微方向性是一阶逻辑题里的常见坑。

定义 3.20：逻辑等价 A ≃ B · 例 ¬∀x.A ≃ ∃x.¬A

PDF 第 60–61 页（累积，讲终态）

定义

A 与 B 逻辑等价，记 A ≃ B，指对任意模型 (M,σ)，M⊨σA 当且仅当 M⊨σB。

⚠ 本讲稿用 ≃（U+2243）表示逻辑等价，对象语言里的等词另用 ≐。Edge 中两个符号可能混在一起，请用 Chrome。

例题验证：¬∀x.A ≃ ∃x.¬A

左边展开：

(¬∀x.A)_M[σ] = B_¬((∀x.A)_M[σ])
             = { F  若 ∀a∈M, A_M[σ[x:=a]] = T
                 T  否则 }

右边展开：

(∃x.¬A)_M[σ] = { T  若 ∃a∈M, (¬A)_M[σ[x:=a]] = T
                 F  否则 }
             = { T  若 ∃a∈M, A_M[σ[x:=a]] = F
                 F  否则 }

两边都是"存在某个 a 让 A[x:=a] 假则 T，否则 F"——逻辑等价 ✓。

德摩根量词律对偶式：¬∃x.A ≃ ∀x.¬A，证法对偶。把否定"穿过"量词时，量词翻面。这是一阶演算化简的常用招。

定理 3.21：等值替换

PDF 第 62 页（章末）

设 B ≃ C。把 A 中的若干 B 替换成 C 得到 A'，则 A ≃ A'。

白话：逻辑等价的子公式可以"无痛替换"——就像数学里 sinx 的某些恒等式可以代入任何含 sinx 的表达式。证明思路：对 A 的结构归纳，每层用 B ≃ C 的语义保持子公式真值。
意义：有了这条定理 + 一些等价律（德摩根、量词分配、双否消去…），就可以做一阶公式的化简 / 规范化，比如转成"前束范式"。这是后续模型论 / 自动定理证明的基础。

常见误解 & 自检题

补充背景 · 收尾前自查

三个最容易栽的坑

坑 1：把 σ[x:=a] 当成"重置 σ"。它只覆盖 x 一处，其他变元的旧值仍在。p33 的例子里 x₁ 还是 1、x₄ 还是 4 就是这个原因。
坑 2：把"A 可满足"和"M⊨A"混用。前者：存在某个 (M,σ) 使 A 真；后者：在指定的 M 上对所有 σ 都真。后者强得多。
坑 3：把 ⊨ 的层级搞混。AM[σ]=T 是单点；⊨A 是普世。看到 ⊨ 先看左侧有没有 Γ、下标有没有 σ。

自检题（不交，但应该会做）

  1. 在 𝒜 标准模型 ℕ, σ(xₙ)=n 下，求 (∃x₂.(x₂ ≐ +(x₁,x₁)))N[σ]。
  2. 判断 ⊨ ∃x.(A∨B) ↔ (∃x.A ∨ ∃x.B) 是否成立。给证明或反例。
  3. 判断 ⊨ ∀x.(A∨B) ↔ (∀x.A ∨ ∀x.B) 是否成立。给证明或反例。
  4. 用引理 3.20 把"Γ∪{A}⊨B"转换为某个不可满足陈述。

参考答案：1. T（取 x₂=2，因 +(x₁,x₁) 算出 2）。2. 成立。3. 不成立（A,B 分别在不同 x 上真）。4. Γ∪{A,¬B} 不可满足。
