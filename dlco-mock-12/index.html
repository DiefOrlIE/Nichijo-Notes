第一部分　二进制信息表示

一、进位计数制与定点数编码

1.1计算
按权展开：(1011.1)₂ = ______（十进制）；十进制 13 转二进制 = ______。
参考答案
11.5；1101。整数除 2 取余倒读，小数乘 2 取整顺读；十六进制 4 位一组、八进制 3 位一组。
1.2计算
8 位补码：[−5]补 = ______；8 位补码表示范围 = ______。
参考答案
1111 1011（+5=0000 0101 取反加一）；[−128, +127]。负数比正数多一个，最小负数 −2ⁿ⁻¹。
1.3解释
补码相比原码，主要好处是什么？（说出两点）
参考答案
① 0 表示唯一（原码有 +0/−0）；② 减法统一为加法、符号位直接参与运算，无需单独判符号 → 硬件简单。
1.4填空
同一数值的移码与补码关系是 ______；移码主要用在 ______。
参考答案
只差最高位（符号位相反）；浮点数的阶码（使指数能按无符号数直接比较大小）。
1.5判断
n 位补码系统的模是 2ⁿ − 1。（对／错）
参考答案
错，模是 2ⁿ。

二、浮点数 IEEE 754

1.6填空
单精度 32 位划分：S ______ 位，E ______ 位（偏置 ______），M ______ 位。
参考答案
1；8（偏置 127）；23。
1.7解释
浮点尾数为什么有个"隐含的 1"？阶码为什么用移码？
参考答案
规格化数尾数恒为 1.xxx，前导 1 固定故不存、省 1 位换精度；阶码用移码使浮点数能按无符号整数顺序直接比较大小。
1.8计算
求单精度 BEE00000H 的真值。
参考答案
−0.4375。S=1 负；E=0111 1101=125→指数 −2；M=.110…=0.75→尾数 1.75；值 = −1.75×2⁻² = −0.4375。
1.9判断
IEEE 754 浮点数的尾数采用补码存储；阶码全 1、尾数全 0 表示 ±∞。（两句都对吗？）
参考答案
前半句错——尾数用原码（移码的是阶码）；后半句对。阶码全 1 尾数≠0=NaN；阶码全 0 尾数≠0=非规格化数。

三、非数值数据与数据类型

1.10解释
C 语言中 -1 < 0U 的结果是真还是假？为什么？
参考答案
假。有符号与无符号比较时有符号被转为无符号，−1 的全 1 位被当作最大无符号数，故 > 0U。
1.11填空
8421 码、余 3 码、格雷码中，无权码且"相邻只差一位"的是 ______；ASCII 是 ______ 位编码、共 ______ 个字符。
参考答案
格雷码；7；128。
1.12解释
汉字机内码如何与 ASCII 码区分？
参考答案
机内码两字节最高位都置 1，而 ASCII 最高位为 0，据此唯一区分。转换链：区位码 →(+32)国标码 →(最高位置1)机内码。

四、数据存储与字节序

1.13计算
0x12345678 在小端机器上从低地址起逐字节存放为 ______。
参考答案
78 56 34 12。小端 = 最低有效字节放低地址。
1.14解释
（题眼）同一段字节，"按整数读"和"按字符串读"，哪个受字节序影响？为什么？
参考答案
整数受字节序影响（整体拆字节后顺序不同）；字符串不受影响，因为它按地址从低到高逐字符读，与端序无关。
第二部分　基于布尔代数的计算组件设计

五、数字抽象与布尔代数

2.1解释
什么是数字抽象？噪声容限的作用是什么？
参考答案
把连续模拟电压划成两段映射为 0/1；噪声容限是"输出保证电平"与"输入识别阈值"之间的余量，保证小干扰不会让 0/1 被误读，使电路可级联、抗噪。
2.2解释
FPGA / PLD 用什么器件实现任意逻辑函数？基本思想是什么？
参考答案
查找表（LUT）：把真值表的输出预存在小存储器里，用输入当地址查出结果 → 硬件功能可由配置决定。
2.3填空
德摩根：¬(X·Y) = ______；¬(X+Y) = ______。
参考答案
¬X+¬Y；¬X·¬Y。分配律两条：X(Y+Z)=XY+XZ；X+YZ=(X+Y)(X+Z)。
2.4计算
化简 XY + ¬X·Z + YZ = ______；被消去的是哪一项？为什么？
参考答案
XY + ¬X·Z；消去 YZ（一致 / 冗余项）。因 YZ=1 时 X=1→XY=1、X=0→¬XZ=1，前两项必有一项已为 1。

六、逻辑函数与卡诺图化简

2.5解释
用自己的话说出 蕴含项、质蕴含项、本质质蕴含项 的区别，以及"最小覆盖"怎么求。
参考答案
蕴含项=卡诺图上任一合法的全 1 圈（对应一个乘积项）；质蕴含项=扩到不能再大的圈；本质质蕴含项=某 1 格只被它一个圈覆盖、必选的圈。最小覆盖=先选所有本质质蕴含项，再用最少的其他质蕴含项补全剩余 1 格。合并 2ⁱ 格消 i 个变量。
2.6判断
卡诺图按格雷码 00 01 11 10 排列，是为了让相邻格只差一个变量；无关项 × 可当 0 也可当 1。（对／错）
参考答案
对。相邻只差一位才能合并消变量；× 哪种能把圈做更大就当哪种。

七、CMOS 实现与组合逻辑组件

2.7填空
CMOS 与非门：下拉 NMOS ______ 联、上拉 PMOS ______ 联（或非门相反）；高态噪声容限 NM_H = ______。
参考答案
串；并；V_OHmin − V_IHmin。口诀：与→下拉串、或→下拉并，PMOS 与 NMOS 对偶；上下拉绝不能同时导通（否则 VDD 到地短路）。
2.8填空
2 选 1 MUX 输出 F = ______；MUX 能否实现任意组合逻辑函数？______；组合逻辑三规则之一是"无 ______"。
参考答案
A·¬S + B·S；能；反馈。
2.9计算
半加器：S = ______，CO = ______；全加器：S = ______，Cout = ______。
参考答案
A⊕B，A·B；A⊕B⊕Cin，AB+(A⊕B)Cin。
2.10解释
行波进位加法器为什么慢？先行进位（CLA）用什么思想解决？（写出"生成 G"和"传播 P"的含义）
参考答案
高位进位必须等低位逐级传来，延迟 ∝ n（约 2n 级门）。CLA 定义 Gᵢ=AᵢBᵢ（本位必产生进位）、Pᵢ=Aᵢ⊕Bᵢ（把低位进位传上去），用 Cᵢ=Gᵢ+PᵢCᵢ₋₁ 展开，使所有进位由输入并行算出、不再逐级等待。

八、竞争冒险

2.11解释
怎么用卡诺图判断电路存在静态冒险？怎么消除？
参考答案
判断：两个为 1 的质蕴含圈"相切而未被同一圈覆盖"，输入跨边界跳变（仅一个变量变）时因路径延迟差出现毛刺。消除：在与或式中加一个冗余的一致项圈搭住相切边界。

九、锁存器与触发器

2.12填空
（必考）锁存器是 ______ 触发，触发器是 ______ 触发；SR 锁存器 S=R=1 是 ______ 态，特征方程 Q* = ______。
参考答案
电平（透明）；边沿；禁止（无效）；Q* = S + ¬R·Q（约束 S·R=0）。
2.13解释
为什么时序电路要用边沿触发的触发器，而不直接用透明的 D 锁存器？
参考答案
D 锁存器使能期间透明，新状态会立刻反馈再次参与运算、一个周期内多次翻转（空翻）；边沿触发只在时钟沿采样一次，状态每周期干净走一步。
2.14计算
① T 触发器现态 Q=1、T=1，次态 Q* = ______，常用于 ______；② JK 触发器现态 Q=0、J=1,K=0，次态 Q* = ______（JK 特征方程 Q*=J·¬Q+¬K·Q）。
参考答案
① 0（Q*=T⊕Q=1⊕1=0），计数器 / 分频；② 1（Q*=1·1+1·0=1）。

十、同步时序设计与时序约束

2.15填空
输出只取决于现态的是 ______ 型，取决于现态和输入的是 ______ 型；设计步骤关键：原始状态图 → 状态化简 → 状态编码 → 选触发器用 ______ 推激励方程 → 画电路。
参考答案
Moore；Mealy；特征方程。
2.16计算
用 T 触发器设计同步模 4 加法计数器（Q1Q0: 00→01→10→11→00），激励方程 T0 = ______，T1 = ______。
参考答案
T0 = 1（最低位每拍都翻）；T1 = Q0（低位为 1 时高位才翻，逢二进一）。
2.17解释
n 个 D 触发器串联（前级 Q 接后级 D）构成什么电路？异步（行波）计数器和同步并行计数器哪个快、为什么？
参考答案
移位寄存器。同步快：异步靠前级输出驱动后级时钟、进位逐级传递有累积延迟；同步所有触发器共时钟同时翻转，无逐级延迟。
2.18计算
建立时间、保持时间分别指什么？设 t_ffpd(max)=3ns、t_comb(max)=5ns、t_setup=1ns，求最小时钟周期与最高频率。把时钟放慢能解决 hold 违例吗？
参考答案
setup=时钟沿前 D 须已稳定的提前量，hold=沿后须继续稳定的滞后量。t_clk > 3+5+1=9ns → 最小周期 9ns、最高 ≈ 111 MHz。不能放慢时钟救 hold：hold 约束 t_hold < t_ffpd(min)+t_comb(min) 不含 t_clk，与时钟无关。

十一、对照真题补漏（去年填空/设计考过、上面没覆盖的）

2.19计算
（真题填空）F(A,B,C,D) 的最小项为 Σm(1,3,5,7,9,11,13,15)，则其最大项为 ΠM(______)；序号最大的最大项 M14 的逻辑表达式 = ______。
参考答案
最大项编号与最小项互补 → ΠM(0,2,4,6,8,10,12,14)；M14：14=1110，最大项是和项、变量为 1 取反变量、为 0 取原变量 → ¬A+¬B+¬C+D。其实这个 F 就是"D=1"（全是奇数项）。
2.20计算
（真题填空）16 个状态至少需要 ______ 位二进制编码；12 位地址的最大可寻址空间为 ______。
参考答案
4 位（⌈log₂16⌉=4）；2¹² = 4096 = 4K 个单元。
2.21解释
（真题填空·解词）CPLD 和 FPGA 分别是什么？基本结构有何不同？
参考答案
CPLD（复杂可编程逻辑器件）：基于乘积项（与-或阵列）结构，规模较小、延迟可预测、多为非易失。FPGA（现场可编程门阵列）：基于查找表 LUT + 可编程互连，规模大、灵活，通常靠 SRAM 配置（掉电丢失）。
2.22解释
（真题填空）状态图的"完备性"指什么？"互斥性"又指什么？
参考答案
完备性：每个状态对输入的所有取值组合都有对应转移（不遗漏）。互斥性：同一状态下任意两条转移的条件不能同时成立（不冲突）。二者保证状态转移确定、无歧义。
2.23解释
（真题设计题）什么是"未用状态分析"？什么样的电路具有"自启动"能力？
参考答案
用 k 位编码 m 个状态时，2ᵏ−m 个用不到的编码就是未用状态。分析方法：假设电路因干扰误入某未用状态，沿状态转移看它能否在有限个时钟内回到有效工作状态。若能（且过程中无错误输出）则电路具有自启动能力；否则需指定未用状态的次态来修正设计。
2.24计算
（真题填空·CMOS 逆向）某 3 输入 CMOS 电路：下拉网络是 3 个 NMOS 串联、上拉是 3 个 PMOS 并联。写出逻辑表达式 F = ______，功能为 ______。
参考答案
F = ¬(A·B·C)；三输入与非门。下拉串联=只有 ABC 全 1 才把输出拉 0 → 与非。给电路反推功能时，盯住下拉网络即可。
