A2函数式编程

一句话主线：函数是一等对象（first-class object）——能绑定到名字、能当参数传、能当返回值、能塞进数据结构。围绕这一点展开高阶函数、闭包、柯里化、装饰器。

① 执行环境：frame 与名字查找

  • 每次函数调用都新建一个独立的 local frame（栈帧）；递归之所以正确，就是每层有自己的 frame。
  • 名字查找顺序：local → 外层（嵌套 parent）→ global（LEGB）。
关键字作用用在哪
无函数内赋值 → 在 local frame 新建绑定默认
global x绑定到 global frame普通/嵌套都行
nonlocal x绑定到最近的非 global 外层 frame仅嵌套函数

只读外层名字不用声明；一旦在函数里赋值且没声明，该名字立刻被当成局部变量，赋值前读取就 UnboundLocalError。

Pure vs Non-Pure：纯函数只靠输入算输出；副作用（改全局/nonlocal、print、改可变参数）就是非纯。可变参数（list/dict）会被函数改到外面，不可变（int/str/tuple）不会。

② 高阶函数：函数作参数 / 作返回值

每次 make_counter() 调用都造一个独立 frame，返回的内层函数保留对它的引用 = 闭包。多个计数器互不干扰，就是因为各有各的 frame。

③ 柯里化 Currying（高频大题结构）

把"一次吃多个参数"拆成"一系列每次吃一个参数、返回新函数"的链条。实现靠闭包，所以柯里化就是闭包的特例。

记忆口诀：f(a)(b)(c) 看成洋葱，每剥一层吃一个参数；函数体里 def 的嵌套层数 = 参数个数。用途：部分应用（先固定一部分参数得到特化版本）、适配只收单参数的高阶函数、延迟绑定。

④ 闭包延迟绑定大坑（必考易错）

Python 闭包捕获的是变量名不是值。循环变量是同一个名字，所有 lambda 共享它，循环结束后都拿到最终值。解法：lambda x, a=cur: ... 用默认参数当场冻结。若内层用 def + 循环变量，则改用工厂函数 make_xxx(a, k) 包一层。

② 继承 · super · 多态

  • 派生类不写 __init__ → 自动继承基类的。
  • 多态：同名方法通过对象调用时按实际类型分派；遍历一个混类型列表 sum(s.area() for s in shapes) 不判断类型，就是多态。

重写 __init__ 却忘记 super().__init__(...) → 基类属性没初始化，后面访问就 AttributeError。

③ MRO 与多继承（高频易错）

super() 不是"父类"，是 MRO（方法解析顺序）里的下一个。 MRO 规则：派生类在前、多基类按声明顺序、C3 线性化保证一致。

想精确调某个基类的方法，绕过 MRO：用类名显式调 Cat.speak(self)。

B3数据规范化

为什么要规范化：各特征量纲/尺度差太大（如"学习时长"和"成绩"），不处理的话大尺度特征会主导距离和方差。两种主流方法 + 除零特判。

方法公式何时用
min-max（缩放到 [0,1]）f' = (f - min) / (max - min)分布有界、要固定范围
z-score（均值0标准差1）f' = (f - μ) / σ有异常值、要消除量纲

max==min（或 σ==0）时分母为 0，必须特判置 0。整数数组先 .astype(float) 防精度丢失。

B4KNN 与距离度量

KNN = 找最近的 K 个邻居投票（分类）或取均值（回归）。是 lazy learning：没有显式训练，预测时才算距离，运行代价高。

距离公式闵氏 p
欧氏 Euclidean√Σ(xᵢ-yᵢ)²p=2
曼哈顿 ManhattanΣ|xᵢ-yᵢ|p=1
切比雪夫 Chebyshevmax|xᵢ-yᵢ|p→∞

K 小 → 贴合数据但易受噪声；K 大 → 边界平滑但钝化；K=全集 → 恒定输出多数类。平票/平距取最小编号是这门课的统一约定。

B5逻辑回归（二分类）

用 sigmoid 把线性输出 z=wᵀx+b 压到 (0,1) 当概率；y≥0.5 判 1。本质是用线性模型预测"对数几率 ln(y/(1-y))=wᵀx+b"。求解 = 极大似然 = 最小化交叉熵，用梯度下降。

梯度更新公式记牢：w -= α·(1/N)Σ(ŷᵢ-yᵢ)xᵢ，b -= α·(1/N)Σ(ŷᵢ-yᵢ)。

B6聚类 · K-Means

无监督：只有 x 没标签，挖数据内在结构。K-Means 目标是最小化簇内平方和 E=Σₖ Σ_{x∈Cₖ} ‖x-μₖ‖²。流程：①初始化 k 个中心 ②每点分到最近中心 ③重算中心=簇内均值 ④重复至不变。

NumPy 版簇心更新一行：centroids[i] = X[labels==i].mean(axis=0)。K-Centroid 变体：簇心必须是某个真实样本点。影响结果的因素：k 值、初始点选择（可能收敛到不同局部解）、噪声。

B7PCA 与 SVD（降维）

PCA 目标：找投影后方差最大的方向。结论链：协方差矩阵的特征值 = 该方向上的方差，特征向量 = 该方向；取最大的几个特征值方向 = 保留信息最多、投影损失最小。

SVD 去噪（保留前 K 个奇异值）

PCA 去噪原理：小特征值对应方向方差小，多是噪声，丢弃即去噪。

为什么先标准化：各特征量纲差异大时，否则大尺度特征主导方差。

eigh 升序：取最大要 np.argsort(vals)[::-1]。XXᵀ 对称 → 特征向量正交 → 各主成分互不相关。

B8关联规则 · 异常检测

① 关联规则 Apriori

  • 支持度 s(X)=σ(X)/N（频繁程度，概率 P(X)）。
  • 置信度 c(X→Y)=σ(X∪Y)/σ(X)（可靠程度，条件概率 P(Y|X)）。⚠ A→B 和 B→A 置信度不同！
  • Apriori 原理：非频繁项集的所有超集必非频繁（反单调性，可剪枝）。等价：X⊆Y ⟹ s(X)≥s(Y)。

判置信度避免浮点误差：用整数比较 100*σ(X∪Y) >= C*σ(X)。
剪枝例：{文档,讨论区} 非频繁 → {文档,讨论区,草稿} 必非频繁，无需再查。

② 异常检测

方法核心
基于模型（3-sigma）z=(x-μ)/σ，|z|>2 或 >3 为异常
基于邻近度density(x,k)=到 K 近邻平均距离，越大越稀疏越异常
基于相对密度（作业用）score(x)=density(x,k) / 邻居们 density 的均值，能处理不同密度簇

异常 ≠ 噪声：噪声随机、观察价值低；异常可能来自不同类别/自然变异/测量误差，有研究价值。np.std 默认 ddof=0（总体）。

B9理论简答速记

这些是简答/选择题的高频答案，背结论。

监督 vs 无监督

  • 监督：数据有标签 (X,y)，学输入→输出映射；任务 = 分类 / 回归。
  • 无监督：只有 X 无标签，学内在结构；任务 = 聚类 / 降维 / 关联规则 / 异常检测。

过拟合 / 欠拟合（loss–复杂度图）

训练 loss 随复杂度持续下降；泛化(测试) loss 先降后升（拐点=最佳复杂度）。图左侧=欠拟合（两者都高），右侧=过拟合（训练低测试高，记住了噪声）。不能只看训练 loss 选模型，它反映不了泛化能力。划分方法：留出法 / 交叉验证。

MSE vs MAE

例（误差 0.5/0.2/-0.5/0.3/-0.5）：MSE=(0.25+0.04+0.25+0.09+0.25)/5≈0.18；MAE=2.0/5=0.40。

list vs ndarray（为何大规模数值计算用 ndarray）

Python listNumPy ndarray
数据任意类型，存指针同类型，固定长度
内存指针散落堆中连续内存块（缓存友好）
计算for 逐个向量化（底层 C/BLAS/SIMD）

IPython 速答

  • In 是 list（存源码字符串），Out 是 dict（key=编号，value=返回值）。只有"光秃秃的表达式"才进 Out；赋值、print()（返回 None）不进。
  • line magic 单 %（仅当前行）；cell magic 双 %%（整个 cell，必须在首行）。%timeit 测耗时、%run 跑脚本。

  5. __iter__ 不重置状态 → 对象只能 for 一次
  6. 可变默认参数 def f(lst=[]) 多次调用共享同一 list → 用 None 哨兵
  7. 二维列表 [[0]*m]*n 所有行同一对象 → 用列表推导
  8. BFS 用 deque.popleft()，别用 list.pop(0)（O(n)）
  9. visited 入队时标记，不是出队时
  10. map/filter 是迭代器，list() 后才能多次遍历
  11. 浮点比较用 abs(a-b)<eps，不用 ==
  12. NumPy 属性不加括号：a.shape 对；但 a.sum() 是方法要括号
  13. axis 记反：axis=0 压行得每列结果，axis=1 压列得每行结果
  14. 切片是视图会改原数组，不想被改用 .copy()
  15. np.where 双分支都执行，除零套 np.errstate
  16. keepdims=True：要和原数组广播相减时必须加
  17. std 的 ddof：默认 0（总体），样本用 ddof=1
  18. SVD 的 S 是一维，重构要 np.diag(S[:K])；eigh 升序取最大要倒序
  19. groupby 后 reset_index()；多条件筛选每个加括号用 & | 不用 and
  20. 平票/平距取最小编号（这门课统一约定）

极简考前自查表

  • ☐ 闭包：make_xxx() 内 def inner() + nonlocal/默认参数 + return
  • ☐ 柯里化：f(a)(b)(c)，def 层数=参数个数
  • ☐ OOP：__init__/super/__str__/__eq__/__add__/__call__
  • ☐ 多态：父类引用调子类重写方法
  • ☐ MRO：Cls.mro()，super 走 MRO
  • ☐ 迭代器：__iter__ 返 self 并重置，__next__ 抛 StopIteration
  • ☐ 异常：try/except/else/finally，raise XxxError
  • ☐ 搜索：BFS 求最短(deque)，DFS/回溯求所有解
  • ☐ 递归：基线 + 递推 + 记忆化 cache
  • ☐ NumPy：广播规则、axis 聚合、视图/副本、where 除零
  • ☐ Pandas：groupby+reset_index、merge(indicator)、Series 按标签对齐
  • ☐ AI：KNN 投票、逻辑回归梯度、K-Means 流程、PCA 取最大特征值
  • ☐ 输入：map(int, input().split())、EOF 终止、exec(input())
  • ☐ 输出：f"{x:.6f}"、"\n".join() 批量
